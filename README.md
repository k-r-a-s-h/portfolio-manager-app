# Portfolio Manager App

This project is a TypeScript Node.js application using LangChain and LangGraph for portfolio management.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

4.  Start production server:
    ```bash
    npm start
    ```
